# 🎓 Annual College Event 2026

A simple and responsive static web page developed using **HTML5** and **CSS3** that provides complete information about the **Annual College Event 2026**.

## 📌 Project Overview

This project demonstrates the use of basic HTML and CSS concepts such as:

- Semantic HTML elements
- Navigation using anchor links
- Ordered and unordered lists
- Text formatting
- CSS styling
- Page layout and colors
- Responsive structure

The webpage contains multiple sections including Event Information, Schedule, Speakers, Registration, and Contact details.

- 🎓 Attractive header with event title
